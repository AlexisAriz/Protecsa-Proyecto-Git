# Portafolio Personal

¡Hola! Soy **Alexis**. Este repositorio es un portafolio personal creado con Markdown. El objetivo de este proyecto es presentar de manera clara y organizada quién soy, cuáles son mis habilidades y qué cosas me interesan.

## Estructura del Proyecto

Este portafolio está compuesto por varios archivos `.md` que puedes explorar:

* **[Perfil.md](Perfil.md):**
    Una breve introducción sobre mí, mis intereses personales y los valores que considero importantes.

* **[Habilidades.md](Habilidades.md):**
    Un desglose de mis habilidades técnicas (lenguajes, tecnologías) y mis habilidades blandas.

* **[Aprendizajes.md](Aprendizajes.md):**
    Una lista de conceptos y comandos clave que he aprendido sobre Git y control de versiones.

* **[Musica.md](Musica.md):**
    Mis gustos musicales y una lista de mis álbumes favoritos.

* **`/Imagenes`:**
    Una carpeta que almacena todas las portadas de los álbumes.

* **`README.md`:**
    ¡Este mismo archivo! La guía principal del repositorio.
